# PR-MESH-BBS Network Online

The Puerto Rico Mesh Bulletin Board System is now operational.

## What is PR-MESH-BBS?

PR-MESH-BBS is a public bulletin board system operating on Meshtastic Channel-0 (LongFast). It provides island-wide communication capabilities for public announcements and situational reports.

## How to Join

1. Configure your Meshtastic device for the PR region
2. Ensure you are on Channel-0 with LongFast modem preset
3. Monitor for broadcasts at 09:00 and 18:00 daily

For more information, visit the PR-CYBR network documentation.
